## Queue

Queue is implemented via a linked list

```javascript
var Algo = require("algo2");
var queue = new Algo.Types.Queue();

for (var index = 0; index < 10; index++)
    queue.enqueue(index);

for (var index = 0; index < 10; index++)
    queue.dequeue();
```
